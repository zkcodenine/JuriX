# JuriX — Sistema de Gestão Jurídica

Sistema completo para gestão de escritórios de advocacia.

## Download

Baixe a versão mais recente na aba **[Releases](https://github.com/zkcodenine/JuriX/releases)**.

## Instalação

1. Baixe o arquivo `JuriX-Setup-x.x.x.exe`
2. Execute o instalador
3. O JuriX será instalado e um atalho criado na área de trabalho

## Atualizações

O JuriX verifica automaticamente por atualizações ao iniciar. Quando uma nova versão estiver disponível, a atualização será baixada e instalada automaticamente.
